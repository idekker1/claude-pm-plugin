# Changelog

## v1.0.0 — 2026-04-19

Initial release. Extracted and generalized from the MLAI/SLAI project.

### Skills included

- `setup` — Bootstrap project structure (new)
- `pm` — Project manager: git workflow, roadmap maintenance, Notion sync (opt-in), code review triggers
- `architect` — Feature design partner: design options, tradeoffs, roadmap updates
- `code-reviewer` — Senior developer: four-pass code review with fix-or-roadmap classification
- `code-guardian` — Consistency watchdog: daily diff checks, comment hygiene, complexity guard
- `pr` — Ship workflow: lint → commit → push → open PR with template enforcement
- `create-job` — Job dispatcher: validates dependency order, roadmap alignment, acceptance criteria
- `run-jobs` — Job orchestrator: lifecycle management, compliance checks, doc sync
- `antigravity` — High-efficiency coding protocol (v1)
- `antigravity2.0` — High-efficiency coding protocol (v2.0)
- `ultimate-protocol` — Zero-English execution protocol

### Templates included

- `ROADMAP.md` — Versioned roadmap with § structure and tech decisions table
- `.ai-agents/REFERENCE.md` — Living technical reference skeleton
- `WORKFLOW.md` — Branch naming, commit format, day-to-day workflow
- `plugin-config.yaml` — Notion opt-in config
- `jobs/README.md` + `jobs/job-template.md` — Job system documentation
- `issues/open.md` + `issues/closed.md` — Issue tracking tables
- `.github/pull_request_template.md` — PR checklist
